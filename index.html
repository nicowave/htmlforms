Wat is a web form?

Form Elements: Control Tags

Control tags are tags that build elements which a user can control to give information. There are four control tags:

  • input
  • select
  • textarea
  • button

input

The default and most common type of input element is the single-line text field. We specify this using the type attribute and set it to "text".

Let's take a closer look at two specific input types: radio and checkboxes.

Radio buttons are used when the user can only select one choice from many. Setting the input's value will allow you to access it using JavaScript, for example.
